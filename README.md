# JupyterHub + Nbgrader Notebooks - Student View

This repository stores learning modules in `Modules_OSPO` and problem sets in `PS_OSPO`.

* `Modules_OSPO` contains notebooks dervied from [`gt-ospo/oss-training`](https://github.com/gt-ospo/oss-training) that have been ***significantly*** modified in order to work correctly with Nbgrader.
* `PS_OSPO` contains problem sets that students can complete as part of an asynchronous training program.

Start with the `Student_Instructions.md` documentation to log in and run the asynchronous training sessions.
